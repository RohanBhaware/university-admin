# 🎓 University Portal — Frontend (Landing Page)

A modern **React.js** landing page for a **University Portal**.  
This project serves as the **frontend interface** or **homepage** of the portal — designed to be responsive, clean, and easily extendable.

---

## 🧭 Overview

This repository contains only the **frontend** of the University Portal, built using **React + Vite**.  
It provides a visually appealing **landing page** that can later be expanded to include authentication, dashboards, and other university features.

---

## 🛠️ Tech Stack

- **React.js** — Component-based frontend library  
- **Vite** — Lightning-fast build tool and dev server  
- **Tailwind CSS** — Styling for layout and responsiveness  
- **JavaScript (ES6+)** — Core scripting language

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/tejas-130704/UniversityPortal.git

# Navigate into the project directory
cd UniversityPortal

# Install dependencies
npm install
# or
yarn install
````

### Run the project

```bash
# Start the development server
npm run dev
# or
yarn dev
```

The app will be running at **[http://localhost:5173/](http://localhost:5173/)** (default Vite port).

### Build for production

```bash
npm run build
# or
yarn build
```
